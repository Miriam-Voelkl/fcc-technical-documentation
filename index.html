Introduction

The gradient CSS data type is a special type of image that consists of a progressive transition between two or more colors.

Linear Gradient

Linear gradients transition colors progressively along an imaginary line. They are generated with the linear-gradient() function. 

.linear-gradient {
      background: linear-gradient(
            to bottom,
            red,
            orange,
            yellow,
            green,
            blue,
            indigo,
            violet
          );
        }
Radial Gradient

Radial gradients transition colors progressively from a center point (origin). They are generated with the radial-gradient() function. 

.radial-gradient {
      background: radial-gradient(
              pink, 
              darkgoldenrod, 
              seagreen);
          }
Conic Gradient

Conic gradients transition colors progressively around a circle. They are generated with the conic-gradient() function. 

.conic-gradient {
      background: conic-gradient(
              violet, 
              indigo, 
              turquoise, 
              indigo);
          }
Repeating Gradients

Repeating gradients duplicate a gradient as much as necessary to fill a given area.

  • Repeating Linear Gradient

    For a repeating linear gradient, the function repeating-linear-gradient() is used. 

    .repeating-linear-gradient { 
      background: repeating-linear-gradient( 
                  to top right, 
                  darkgoldenrod,
                  darkgoldenrod 5px, 
                  pink 5px, 
                  pink 10px ); 
                }

  • Repeating Radial Gradient

    Repeating gradients duplicate a gradient as much as necessary to fill a given area. For a repeating radial gradient, the function repeating_radial_gradient()is used. 

    .repeating-radial-gradient {
      background: repeating-radial-gradient(
                  indigo,
                  indigo 15px,
                  white 15px,
                  white 30px
                );
              }

  • Repeating Conic Gradient

    Repeating gradients duplicate a gradient as much as necessary to fill a given area. For a repeating conic gradient, the function repeating_conic_gradient() is used. 

    .repeating-conic-gradient {
      background: repeating-conic-gradient(
                seagreen, pink 20deg);
            }
